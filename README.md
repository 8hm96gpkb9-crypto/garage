# Папка для фотографий товаров

Сюда нужно положить фотографии вещей, которые ты продаёшь.

## Как использовать

1. Скопируй сюда свои фото — можно называть как угодно, например:
   `kurtka1.jpg`, `kurtka2.jpg`, `polka.jpg` и т.д.
   Лучше заранее сжать фото (до ~500 КБ - 1 МБ каждое), чтобы каталог быстро грузился.

2. Открой `../index.html`, найди массив `PRODUCTS` и в нужном товаре укажи путь к фото
   в поле `photos`, например:

   ```js
   {
     id: 1,
     name: "Кожаная куртка, размер M",
     price: "3500 ₽",
     photos: ["images/kurtka1.jpg", "images/kurtka2.jpg"],
     description: "...",
     reserved: false
   }
   ```

   Первое фото в списке будет показываться на карточке в каталоге,
   остальные — доступны для просмотра в самой карточке товара (миниатюры).

3. Если для товара оставить `photos: []` (пусто) — вместо фото будет
   цветная плашка с первой буквой названия (как в демо-варианте).

## Важно при заливке на GitHub

Когда будешь создавать репозиторий и заливать туда `webapp/`, не забудь
загрузить и эту папку `images` вместе с файлами фото внутри — GitHub не
создаёт пустые папки сам по себе, но как только в ней появится хотя бы
один файл, она подтянется вместе с остальным.
